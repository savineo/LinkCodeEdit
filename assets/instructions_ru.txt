LINKCODEEDIT — справка

• Вкладка «Комментарии»: удаляет комментарии из кода. Режим ALL — удаляем из HTML/CSS/JS/PHP/JSON/Python.
• «Форматирование»: форматирует или минифицирует. JSON умеет сортировать ключи.
• «Обфускация»: eval(Base64) / hex+eval / Base64-контейнер. Можно добавить комментарий и протестировать JS в браузере.
• «ЧПУ URL (slug)»: генерирует slug из текста (транслитерация ru→lat, регистр, разделитель, максимум длины).
• «Количество символов»: быстрый подсчёт символов, слов, строк, пробелов, и т.д.
• «UTM-метки»: конструктор ссылок с UTM; можно перезаписывать существующие метки.
• «JS-ссылка»: генератор закодированной ссылки (например, для вставки на страницу).
• «Генератор паролей»: множество паролей по заданной маске.
• «Пакетная обработка»: форматирование/минимизация файлов папки (настройки слева внизу).

Горячие клавиши: Ctrl+Enter — выполнить действие вкладки; меню «Правка» — обмен вход↔выход, очистка, буфер.
